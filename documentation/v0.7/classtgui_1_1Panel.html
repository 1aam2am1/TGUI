Detailed Description

Panel widget.

The panel is a static container of widgets. The background color can be solid or transparent. It can be used for having radio button groups or any other reason to group certain widgets.

Signals:

  • MousePressed (the left mouse button was pressed on top of the panel)
    • Optional parameter sf::Vector2f: Mouse position relative to the panel position
    • Uses Callback member 'mouse'
  • MouseReleased (the left mouse button was released on top of the panel)
    • Optional parameter sf::Vector2f: Mouse position relative to the panel position
    • Uses Callback member 'mouse'
  • Clicked (you left clicked the panel)
    • Optional parameter sf::Vector2f: Mouse position relative to the panel position
    • Uses Callback member 'mouse'
  • Inherited signals from Container

Changes the transparency of the widget.

Parameters
transparencyThe transparency of the widget. 0 is completely transparent, while 255 (default) means fully opaque.

Note that this will only change the transparency of the images. The parts of the widgets that use a color will not be changed. You must change them yourself by setting the alpha channel of the color.
