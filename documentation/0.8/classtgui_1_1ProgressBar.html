Detailed Description

Progress bar widget.

Signals:

  • ValueChanged (The value of the progress bar has changed)
    • Optional parameter int: New value
    • Uses Callback member 'value'
  • Full (The new value equals the maximum value)
    • Optional parameter int: The maximum value
    • Uses Callback member 'value'
  • Inherited signals from ClickableWidget

§ FillDirection

enum tgui::ProgressBar::FillDirection
strong

The fill direction of the progress bar.

Determines in which direction the progress bar is filled when value goes from minimum to maximum.

Enumerator
LeftToRight 

Start filling at the left side and go to the right.

RightToLeft 

Start filling at the right side and go to the left.

TopToBottom 

Start filling at the top an go downward.

BottomToTop 

Start filling at the bottom and go upward.
