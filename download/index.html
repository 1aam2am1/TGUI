What version should you download?
TGUI v0.6 comes with a tons of new features and TGUI v0.5 is already more than a year old, so I am recommending TGUI v0.6.
Note that v0.6 requires c++11, so if don’t have VC++10 or GCC 4.6 or newer then you will have to use v0.5.

TGUI v0.6 – development version (RECOMMENDED)
Download TGUI v0.6 RC (with precompiled MinGW 4.7 TDM (SJLJ) 32 bits libraries for SFML 2.1)
Download TGUI v0.6 RC (with precompiled Visual C++ 11 (2012) 32 bits libraries for SFML 2.1)
Download the latest development snapshot from github.

TGUI v0.5 – stable version
Download TGUI v0.5.2 (with precompiled MinGW-4.7.1-TDM 32bit libraries for SFML 2.1)
Download TGUI v0.5.2 (source code to build for your own compiler for SFML 2.x)
