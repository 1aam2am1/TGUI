TGUI v0.7-alpha3

Development version, recommended for most projects. TGUI 0.7 is in alpha stage meaning that the biggest rewrites have already taken place, but API breaking changes will still occur. Precompiled libraries require SFML 2.3.2. Note that the 32bit and 64bit are about the compiler you use, not about your windows.
